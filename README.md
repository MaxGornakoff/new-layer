# Filament Shop

Интернет-магазин филамента для 3D-печати: Vue 3 + Laravel API + MySQL.

## Стек

- **Frontend:** Vue 3, Vue Router, Pinia, Vite, Tailwind CSS v4
- **Backend:** Laravel 13, Sanctum (API tokens)
- **БД:** MySQL 8.4
- **Локально:** Docker (nginx + PHP 8.4 + MySQL)

## Структура

```
backend/     Laravel API (document root: backend/public)
frontend/    Vue SPA
docker/      nginx и PHP Dockerfile
```

Один домен в продакшене: Vue собирается в `backend/public`, API — под `/api`.

## Быстрый старт

### 1. Backend (.env)

```bash
cp backend/.env.docker backend/.env
```

Сгенерируйте ключ (через Docker после шага 2):

```bash
docker compose run --rm php php artisan key:generate
```

### 2. Docker

```bash
docker compose up -d --build
docker compose exec php php artisan migrate --seed
```

API: http://localhost:8080/api/health

### 3. Frontend (разработка)

```bash
cd frontend
npm install
npm run dev
```

Сайт: http://localhost:5173 (прокси `/api` → `:8080`)

### 4. Production-сборка фронта

```bash
cd frontend
npm run build
```

Статика попадёт в `backend/public` (рядом с `index.php` Laravel).

## API (основное)

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/categories` | Категории (ABS, PLA, …) |
| GET | `/api/hero-slides` | Слайды главной (+ `meta`: автопрокрутка, интервал) |
| GET | `/api/site-settings` | Логотип, фавикон, настройки каталога и слайдера |
| GET | `/api/products?search=&category=&color=` | Каталог с фильтрами |
| POST | `/api/register`, `/api/login` | Авторизация |
| POST | `/api/orders` | Оформление заказа (auth) |
| GET | `/api/orders` | Список заказов клиента с позициями (auth) |
| GET | `/api/admin/products` | Товары (admin) |
| GET/POST | `/api/admin/hero-slides` | Слайды главной (admin) |
| POST | `/api/admin/site-settings` | Логотип, фавикон, настройки (admin) |
| POST | `/api/admin/products/{id}/stock` | Пополнение остатков (admin) |

Заголовок авторизации: `Authorization: Bearer {token}`

## Деплой на хостинг

1. Собрать фронт: `npm run build` в `frontend/`
2. Залить `backend/` на сервер (включая `vendor/`, если нет Composer)
3. Document root → `backend/public`
4. Настроить `.env` (MySQL, `APP_URL`, `APP_DEBUG=false`)
5. Выполнить `php artisan migrate --seed` и `php artisan storage:link`

## Текущий фокус и отложенные задачи

**Сейчас в приоритете (следующая сессия):**
- **Доставка:** интеграция калькуляции по API со службами (Байкал Сервис, СДЭК, ПЭК, Яндекс Доставка и др.) — отдельный сервисный слой в PHP
- **Админка — грузоместо:** раздел с параметрами упаковки на **10 катушек** (ширина, длина, высота, вес) для расчёта тарифов
- Доработка превью карточек товаров в каталоге (по возможности параллельно или после доставки)
- Полировка главной: секция «О нас» и прочие блоки

**Отложено на потом (углублённые доработки):**
- Оформление заказа: сверка цен/остатков с сервером на checkout
- Страница «Заказ оформлен» с номером заказа
- Прочие улучшения воронки покупки (гость vs авторизация, полировка UX)

**Уже работает:**
- **Главная (hero):** слайдер под шапкой из админки (`/admin/hero-slides`), HTML в заголовке/подзаголовке, кнопка со стрелкой, свайп и перетаскивание мышью, пагинация-точки, автопрокрутка с интервалом из админки, зацикливание
- **Главная (цифры):** блок `HeroDigits` поверх слайдера — анимация набегания цифр от 0 при появлении в viewport
- **Главная (категории):** блок `HomeCategories` — карточки по макету, преимущества, градиентный фон из 2 цветов (админка), белый круг 20%, клик → каталог с фильтром
- **Стилизация:** правило Tailwind-first (`.cursor/rules/frontend-tailwind.mdc`) — утилиты в приоритете, scoped CSS только по необходимости
- Динамическое меню из админки, поиск в шапке, мини-корзина и меню профиля по hover (десктоп)
- **Корзина и количество:** `+` / `−` / поле ввода синхронизируются с корзиной напрямую (без отдельной кнопки «В корзину»); debounce 1 с при ручном вводе + мгновенное применение по blur/Enter; правило заказа кратно 10 катушкам (сумма по всей корзине) — на фронте и в API
- Корзина: тосты, `localStorage`, лимит по остатку, страница корзины, мини-корзина, базовый checkout
- **Личный кабинет:** «Мои заказы» — раскрываемые карточки с составом заказа, количествами и ссылками на товары
- Прелоадеры (`AppLoader`) на страницах с загрузкой данных с API
